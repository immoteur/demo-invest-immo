# Security Policy

## Supported Versions

We support the latest main branch and the most recent release tag. Security
fixes will be released as soon as possible.

## Reporting a Vulnerability

If you discover a security issue, please email support@immoteur.com with a
clear description and steps to reproduce. Please do not open a public issue for
security reports.

We will acknowledge receipt within 72 hours and provide a timeline for a fix.
